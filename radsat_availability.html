URL Parameters

You can bookmark specific views using URL parameters:

  • config - Set the config (e.g., radsat_image)
  • station - Set the station (e.g., TMS, TCR, H8/9)
  • date - Set the date in YYYY-MM-DD format

Example: ?config=radsat_image&station=H8/9&date=2026-01-15
